Recentemente, a Estácio tornou obrigatório a leitura de seus conteúdos, que são disponibilizados para estudantes. Eu entendo que isso é uma forma de aumentar o uso da plataforma por parte da gente, mas, há alguns como eu, que utiliza dos PDFs que eles disponibilizam e, com ajuda de ferramentas de aprendizagem de máquina, conseguimos resumos eficientes e precisos para estudar. Dessa forma, esperar 15 minutos se torna inviável e, pensando numa forma fácil de contornar isso, eu criei esse programa em rust.

Primeiro, você precisa ser estudante da Universidade, isso é óbvio. Segundo, você precisa ter a noção básica de pegar seu próprio bearer token para acessar as rotas da API deles.

Por sinal, as rotas são "públicas" pelo próprio navegador e nada aqui tem o intuito de prejudicar a Universidade, afinal, até mesmo eu, como estudante deles, adquiri bastante conhecimento com essa brincadeira. 

Uma forma de solução para eles que fazerem a verificação do tempo no lado do backend ao invés do frontend. Dessa forma, esse programa perde sua utilidade. Até lá, espero que usufruam dessa ferramenta.
